# Dotfiles
My personal configuration files

# Installation

### This script will reset existing configuration by removing configuration files: 
- ~/.config/i3
- ~/.config/i3-status
- ~/.config/dunst
- ~/.tmux.conf
- ~/.bashrc 
- ~/.emacs.d
- ~/.gitconfig
- ~/.vimrc

```sh
curl https://raw.githubusercontent.com/ferhadme/dotfiles/refs/heads/master/install.sh | sh
```
